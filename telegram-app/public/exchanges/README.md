# Exchange icons

Place exchange logo files in this folder so the Trade tab can render real icons.

Expected filenames (SVG preferred, PNG also works):

- `binance.svg`
- `bybit.svg`
- `okx.svg`
- `coinbase.svg`
- `kraken.svg`
- `kucoin.svg`
- `bitget.svg`
- `gate.svg` (Gate.io)
- `mexc.svg`
- `htx.svg` (HTX / Huobi)

Notes:
- Keep icons square-ish for best results.
- Use transparent backgrounds if possible.
- If an icon file is missing or fails to load, the UI will fall back to initials.
