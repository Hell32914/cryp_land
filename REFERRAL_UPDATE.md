# Реферальная система - обновление

## Изменения

### 1. Активация рефералов при $1000
- Добавлено поле `isActiveReferral` в модель User
- Реферал становится **активным** только когда его `totalDeposit >= $1000`
- До активации реферал считается **теневым** - не приносит доход

### 2. Автоматическое уведомление
- Когда реферал активируется, реферер получает уведомление в боте
- Начисление 4%/3%/2% начинается только после активации

### 3. Миграция базы данных

После деплоя на Railway нужно выполнить:

```bash
node migrate-referral-active.cjs
```

Это установит `isActiveReferral = true` для всех существующих пользователей с депозитом >= $1000.

### 4. Whitepaper в модальном окне

- Whitepaper теперь открывается во всплывающем окне при клике на "Whitepaper" в хедере
- Легко просмотреть и закрыть
- Удалён как отдельная секция со страницы

## Логика работы

1. Пользователь A приглашает пользователя B по реферальной ссылке
2. B становится **теневым рефералом** A (записано в `referredBy`)
3. Когда B пополняет баланс до $1000, он становится **активным рефералом**
4. A получает уведомление об активации
5. С этого момента A получает 4% от дневной прибыли B
6. Если у A есть реферер (C), то C получает 3% от прибыли B
7. Если у C есть реферер (D), то D получает 2% от прибыли B
